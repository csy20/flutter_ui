# SnackBar in Flutter Material UI

## What is SnackBar?

**SnackBar** is a lightweight message widget that appears temporarily at the bottom of the screen to provide brief feedback about an operation. It's part of Material Design's feedback system and is commonly used to show status messages, confirmations, or simple notifications without interrupting the user's workflow.

## Key Features

- **Temporary Display**: Automatically disappears after a set duration
- **Bottom Positioning**: Appears at the bottom of the screen
- **Action Support**: Can include action buttons for user interaction
- **Queue Management**: Multiple SnackBars are queued automatically
- **Customizable**: Colors, duration, content, and behavior
- **Accessibility**: Screen reader support and semantic announcements
- **Material Design**: Follows Material Design guidelines

## When to Use SnackBar

### ✅ Good Use Cases
- Confirming actions (e.g., "Message sent", "Item deleted")
- Showing brief status updates
- Providing undo functionality
- Displaying network status messages
- Quick feedback for user actions

### ❌ Avoid SnackBar For
- Critical error messages (use Dialog instead)
- Long-form content (use Modal or separate screen)
- Permanent information display
- Complex user interactions

```dart
import 'package:flutter/material.dart';

class BasicSnackBarExample extends StatelessWidget {

                        SizedBox(width: 8),

                    ),
                    backgroundColor: Colors.green,
                  ),
                );
              },
              child: Text('Show Success SnackBar'),

            // SnackBar with Action

                ScaffoldMessenger.of(context).showSnackBar(
                  SnackBar(
                    content: Text('Item deleted'),
                    action: SnackBarAction(
                      label: 'UNDO',
                      onPressed: () {
                        // Handle undo action
                        ScaffoldMessenger.of(context).showSnackBar(
                          SnackBar(
                            content: Text('Item restored'),
                            backgroundColor: Colors.green,
                          ),
                        );
                      },
                    ),
                  ),
                );
              },
              child: Text('Show SnackBar with Action'),
            ),
            
            SizedBox(height: 10),
            
            // Error SnackBar
            ElevatedButton(
              onPressed: () {
                ScaffoldMessenger.of(context).showSnackBar(
                  SnackBar(
                    content: Row(
                      children: [
                        Icon(Icons.error, color: Colors.white),
                        SizedBox(width: 8),
                        Expanded(
                          child: Text('Error: Something went wrong'),
                        ),
                      ],
                    ),
                    backgroundColor: Colors.red,
                    action: SnackBarAction(
                      label: 'RETRY',
                      textColor: Colors.white,

                      },
                    ),
                  ),
                );
              },
              child: Text('Show Error SnackBar'),
            ),
          ],
        ),
      ),
    );
  }
```

```dart
class AdvancedSnackBarExample extends StatefulWidget {
  @override
  _AdvancedSnackBarExampleState createState() => _AdvancedSnackBarExampleState();

class _AdvancedSnackBarExampleState extends State<AdvancedSnackBarExample> {
  int _messageCount = 0;
  List<String> _undoStack = [];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Advanced SnackBar Features'),
        backgroundColor: Colors.indigo,
      ),
      body: Padding(
        padding: EdgeInsets.all(16.0),

              'Advanced SnackBar Features',

            // Queue Management
            ElevatedButton(
              onPressed: () {
                _showQueuedMessages(context);
              },
              child: Text('Show Multiple SnackBars (Queue)'),
            ),
            
            SizedBox(height: 10),
            
            // Progress SnackBar
            ElevatedButton(
              onPressed: () {
                _showProgressSnackBar(context);
              },
              child: Text('Show Progress SnackBar'),
            ),
            
            SizedBox(height: 10),
            
            // Persistent SnackBar
            ElevatedButton(
              onPressed: () {
                _showPersistentSnackBar(context);

            // Interactive SnackBar
            ElevatedButton(

                _showInteractiveSnackBar(context);
              },

            ),
            
            SizedBox(height: 10),
            
            // Network Status SnackBar
            ElevatedButton(
              onPressed: () {
                _showNetworkStatusSnackBar(context);
              },
              child: Text('Simulate Network Status'),
            ),
            
            SizedBox(height: 10),
            
            // Form Validation SnackBar
            ElevatedButton(
              onPressed: () {
                _showValidationSnackBar(context);
              },
              child: Text('Show Validation SnackBar'),
            ),
            
            SizedBox(height: 20),
            
            Text(
              'Message Count: $_messageCount',
              style: TextStyle(fontSize: 16),
            ),
            
            if (_undoStack.isNotEmpty) ...[
              SizedBox(height: 10),
              Text(
                'Undo Stack: ${_undoStack.length} items',
                style: TextStyle(fontSize: 14, color: Colors.grey[600]),
              ),
            ],
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () {
          ScaffoldMessenger.of(context).clearSnackBars();
        },
        child: Icon(Icons.clear_all),
        tooltip: 'Clear all SnackBars',
      ),
    );
  }

  void _showQueuedMessages(BuildContext context) {
    final messages = [
      'First message',
      'Second message',
      'Third message',
      'Final message',
    ];

    for (int i = 0; i < messages.length; i++) {
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(
          content: Text(messages[i]),
          duration: Duration(seconds: 2),
          backgroundColor: Colors.blue[600 + (i * 100)],
        ),
      );
    }
  }

  void _showProgressSnackBar(BuildContext context) {
    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: Row(
          children: [
            SizedBox(
              width: 20,
              height: 20,
              child: CircularProgressIndicator(
                strokeWidth: 2,
                valueColor: AlwaysStoppedAnimation<Color>(Colors.white),
              ),
            ),
            SizedBox(width: 16),
            Text('Processing your request...'),
          ],
        ),
        duration: Duration(seconds: 4),
        backgroundColor: Colors.orange[700],
      ),
    );

    // Simulate completion
    Future.delayed(Duration(seconds: 4), () {
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(
          content: Row(
            children: [
              Icon(Icons.check_circle, color: Colors.white),
              SizedBox(width: 8),
              Text('Request completed successfully!'),
            ],
          ),
          backgroundColor: Colors.green,
          duration: Duration(seconds: 2),
        ),
      );
    });
  }

  void _showPersistentSnackBar(BuildContext context) {
    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: Text('This message persists until dismissed'),
        duration: Duration(days: 1), // Very long duration
        action: SnackBarAction(
          label: 'DISMISS',
          onPressed: () {
            ScaffoldMessenger.of(context).hideCurrentSnackBar();
          },
        ),
        backgroundColor: Colors.purple[700],
      ),
    );
  }

  void _showInteractiveSnackBar(BuildContext context) {
    setState(() {
      _messageCount++;
      _undoStack.add('Message $_messageCount');
    });

    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: Text('Message $_messageCount sent'),
        action: SnackBarAction(
          label: 'UNDO',
          onPressed: () {
            setState(() {
              if (_undoStack.isNotEmpty) {
                String undoneMessage = _undoStack.removeLast();
                _messageCount--;
                
                ScaffoldMessenger.of(context).showSnackBar(
                  SnackBar(
                    content: Text('Undone: $undoneMessage'),
                    backgroundColor: Colors.grey[700],
                    duration: Duration(seconds: 2),
                  ),
                );
              }
            });
          },
        ),
        backgroundColor: Colors.blue[700],
      ),
    );
  }

  void _showNetworkStatusSnackBar(BuildContext context) {
    // Show disconnected
    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: Row(
          children: [
            Icon(Icons.wifi_off, color: Colors.white),
            SizedBox(width: 8),
            Text('No internet connection'),
          ],
        ),
        backgroundColor: Colors.red[700],
        duration: Duration(seconds: 3),
      ),
    );

    // Simulate reconnection
    Future.delayed(Duration(seconds: 3), () {
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(
          content: Row(
            children: [
              Icon(Icons.wifi, color: Colors.white),
              SizedBox(width: 8),
              Text('Back online'),
            ],
          ),
          backgroundColor: Colors.green[700],
          duration: Duration(seconds: 2),
        ),
      );
    });
  }

  void _showValidationSnackBar(BuildContext context) {
    final errors = [
      'Email is required',
      'Password must be at least 8 characters',
      'Please accept terms and conditions',
    ];

    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: Column(
          mainAxisSize: MainAxisSize.min,
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text(
              'Form validation errors:',
              style: TextStyle(fontWeight: FontWeight.bold),
            ),
            SizedBox(height: 4),
            ...errors.map((error) => Padding(
              padding: EdgeInsets.only(left: 8, top: 2),
              child: Text('• $error', style: TextStyle(fontSize: 12)),
            )),
          ],
        ),
        backgroundColor: Colors.red[700],
        duration: Duration(seconds: 5),
        action: SnackBarAction(
          label: 'FIX',
          textColor: Colors.white,
          onPressed: () {
            // Navigate to form or highlight errors
          },
        ),
      ),
    );
  }
}
```

```dart
// Model for different message types
enum MessageType {
  success,
  error,
  warning,
  info,
  loading,
}

class AppMessage {
  final String text;
  final MessageType type;
  final String? actionLabel;
  final VoidCallback? onActionPressed;
  final Duration? duration;

  AppMessage({
    required this.text,
    required this.type,
    this.actionLabel,
    this.onActionPressed,
    this.duration,
  });
}

class SnackBarService {
  static final Map<MessageType, Color> _typeColors = {
    MessageType.success: Colors.green,
    MessageType.error: Colors.red,
    MessageType.warning: Colors.orange,
    MessageType.info: Colors.blue,
    MessageType.loading: Colors.grey,
  };

  static final Map<MessageType, IconData> _typeIcons = {
    MessageType.success: Icons.check_circle,
    MessageType.error: Icons.error,
    MessageType.warning: Icons.warning,
    MessageType.info: Icons.info,
    MessageType.loading: Icons.hourglass_empty,
  };

  static void show(BuildContext context, AppMessage message) {
    Widget content;
    
    if (message.type == MessageType.loading) {
      content = Row(
        children: [
          SizedBox(
            width: 20,
            height: 20,
            child: CircularProgressIndicator(
              strokeWidth: 2,
              valueColor: AlwaysStoppedAnimation<Color>(Colors.white),
            ),
          ),
          SizedBox(width: 16),
          Expanded(child: Text(message.text)),
        ],
      );
    } else {
      content = Row(
        children: [
          Icon(_typeIcons[message.type], color: Colors.white),
          SizedBox(width: 8),
          Expanded(child: Text(message.text)),
        ],
      );
    }

    SnackBarAction? action;
    if (message.actionLabel != null && message.onActionPressed != null) {
      action = SnackBarAction(
        label: message.actionLabel!,
        textColor: Colors.white,
        onPressed: message.onActionPressed!,
      );
    }

    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: content,
        backgroundColor: _typeColors[message.type],
        behavior: SnackBarBehavior.floating,
        margin: EdgeInsets.all(16),
        shape: RoundedRectangleBorder(
          borderRadius: BorderRadius.circular(8),
        ),
        action: action,
        duration: message.duration ?? _getDefaultDuration(message.type),
      ),
    );
  }

  static Duration _getDefaultDuration(MessageType type) {
    switch (type) {
      case MessageType.error:
        return Duration(seconds: 5);
      case MessageType.loading:
        return Duration(seconds: 30);
      case MessageType.success:
      case MessageType.info:
      case MessageType.warning:
      default:
        return Duration(seconds: 3);
    }
  }
}

            context,
            AppMessage(
              text: '$deletedItem restored',
              type: MessageType.success,
            ),
          );
        },
      ),
    );
  }
```

## Tips and Best Practices

### 1. **Message Clarity**
- Keep messages concise and actionable
- Use clear, user-friendly language
- Avoid technical jargon

### 2. **Visual Design**
- Use consistent colors for message types
- Include relevant icons for better recognition
- Consider floating behavior for better UX

### 3. **User Experience**
- Provide undo functionality for destructive actions
- Show progress for long-running operations
- Queue messages appropriately

### 4. **Performance**
- Clear unnecessary SnackBars to prevent memory leaks
- Use ScaffoldMessenger properly
- Avoid showing too many messages simultaneously

### 5. **Accessibility**
- Ensure sufficient color contrast
- Provide semantic announcements
- Support screen readers

## Common Use Cases

### 1. **Form Feedback**
- Validation errors and success messages
- Save confirmations
- Field-specific warnings

### 2. **Network Operations**
- API call status updates
- Connection status changes
- Data synchronization feedback

### 3. **User Actions**
- Delete confirmations with undo
- Copy/share success messages
- Settings change confirmations

### 4. **System Notifications**
- App updates available
- Storage warnings
- Permission requests

SnackBar is an essential component for providing user feedback in Flutter applications. When implemented with proper styling, timing, and accessibility considerations, it creates a smooth and informative user experience without being intrusive or overwhelming.
