# Progress Indicators in Flutter Material UI

## What are Progress Indicators?

**Progress Indicators** are essential UI components that communicate the status of ongoing processes to users. They provide visual feedback about loading states, file uploads, downloads, form submissions, and other time-consuming operations. Flutter Material UI provides two main types: CircularProgressIndicator and LinearProgressIndicator.

## Types of Progress Indicators

### 1. CircularProgressIndicator
- Circular spinning indicator
- Best for unknown duration tasks
- Compact and space-efficient

### 2. LinearProgressIndicator
- Horizontal bar indicator
- Shows progress as a percentage
- Better for known duration tasks

## Key Features

- **Determinate and Indeterminate**: Support for both known and unknown progress
- **Customizable**: Colors, sizes, and styling options
- **Animated**: Smooth animations for better UX
- **Themeable**: Integrates with Material Design theme
- **Accessible**: Screen reader support
- **Performance Optimized**: Efficient rendering

## Animated Progress Indicators

### Dynamic Progress with Animation

```dart
class AnimatedProgressIndicatorExample extends StatefulWidget {
  @override
  _AnimatedProgressIndicatorExampleState createState() =>
      _AnimatedProgressIndicatorExampleState();
}

class _AnimatedProgressIndicatorExampleState
    extends State<AnimatedProgressIndicatorExample>
    with TickerProviderStateMixin {
  late AnimationController _circularController;
  late AnimationController _linearController;
  late Animation<double> _circularAnimation;
  late Animation<double> _linearAnimation;
  
  double _downloadProgress = 0.0;
  bool _isDownloading = false;

  @override
  void initState() {
    super.initState();
    
    _circularController = AnimationController(
      duration: Duration(seconds: 3),
      vsync: this,
    );
    
    _linearController = AnimationController(
      duration: Duration(seconds: 2),
      vsync: this,
    );
    
    _circularAnimation = Tween<double>(
      begin: 0.0,
      end: 1.0,
    ).animate(CurvedAnimation(
      parent: _circularController,
      curve: Curves.easeInOut,
    ));
    
    _linearAnimation = Tween<double>(
      begin: 0.0,
      end: 1.0,
    ).animate(CurvedAnimation(
      parent: _linearController,
      curve: Curves.linear,
    ));
  }

  void _startCircularAnimation() {
    _circularController.forward().then((_) {
      _circularController.reset();
    });
  }

  void _startLinearAnimation() {
    _linearController.forward().then((_) {
      _linearController.reset();
    });
  }

  void _simulateDownload() {
    setState(() {
      _isDownloading = true;
      _downloadProgress = 0.0;
    });

    Timer.periodic(Duration(milliseconds: 100), (timer) {
      setState(() {
        _downloadProgress += 0.01;
        if (_downloadProgress >= 1.0) {
          _downloadProgress = 1.0;
          _isDownloading = false;
          timer.cancel();
        }
      });
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Animated Progress Indicators'),
        backgroundColor: Colors.teal,
      ),
      body: Padding(
        padding: EdgeInsets.all(16.0),
        child: Column(
          children: [
            // Animated Circular Progress
            Card(
              child: Padding(
                padding: EdgeInsets.all(20),
                child: Column(
                  children: [
                    Text(
                      'Animated Circular Progress',
                      style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
                    ),
                    SizedBox(height: 20),
                    AnimatedBuilder(
                      animation: _circularAnimation,
                      builder: (context, child) {
                        return Column(
                          children: [
                            CircularProgressIndicator(
                              value: _circularAnimation.value,
                              strokeWidth: 6.0,
                              valueColor: AlwaysStoppedAnimation<Color>(Colors.teal),
                            ),
                            SizedBox(height: 10),
                            Text('${(_circularAnimation.value * 100).toInt()}%'),
                          ],
                        );
                      },
                    ),
                    SizedBox(height: 20),
                    ElevatedButton(
                      onPressed: _startCircularAnimation,
                      child: Text('Start Circular Animation'),
                    ),
                  ],
                ),
              ),
            ),
            
            SizedBox(height: 20),
            
            // Animated Linear Progress
            Card(
              child: Padding(
                padding: EdgeInsets.all(20),
                child: Column(
                  children: [
                    Text(
                      'Animated Linear Progress',
                      style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
                    ),
                    SizedBox(height: 20),
                    AnimatedBuilder(
                      animation: _linearAnimation,
                      builder: (context, child) {
                        return Column(
                          children: [
                            LinearProgressIndicator(
                              value: _linearAnimation.value,
                              minHeight: 8.0,
                              backgroundColor: Colors.grey[300],
                              valueColor: AlwaysStoppedAnimation<Color>(Colors.blue),
                            ),
                            SizedBox(height: 10),
                            Text('${(_linearAnimation.value * 100).toInt()}%'),
                          ],
                        );
                      },
                    ),
                    SizedBox(height: 20),
                    ElevatedButton(
                      onPressed: _startLinearAnimation,
                      child: Text('Start Linear Animation'),
                    ),
                  ],
                ),
              ),
            ),
            
            SizedBox(height: 20),
            
            // Simulated Download Progress
            Card(
              child: Padding(
                padding: EdgeInsets.all(20),
                child: Column(
                  children: [
                    Text(
                      'Download Simulation',
                      style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
                    ),
                    SizedBox(height: 20),
                    Row(
                      children: [
                        Expanded(
                          child: LinearProgressIndicator(
                            value: _downloadProgress,
                            minHeight: 10.0,
                            backgroundColor: Colors.grey[300],
                            valueColor: AlwaysStoppedAnimation<Color>(
                              _downloadProgress == 1.0 ? Colors.green : Colors.orange,
                            ),
                          ),
                        ),
                        SizedBox(width: 10),
                        Text('${(_downloadProgress * 100).toInt()}%'),
                      ],
                    ),
                    SizedBox(height: 10),
                    Text(
                      _isDownloading 
                        ? 'Downloading...' 
                        : _downloadProgress == 1.0 
                          ? 'Download Complete!' 
                          : 'Ready to download',
                      style: TextStyle(
                        color: _downloadProgress == 1.0 ? Colors.green : Colors.grey[600],
                      ),
                    ),
                    SizedBox(height: 20),
                    ElevatedButton(
                      onPressed: _isDownloading ? null : _simulateDownload,
                      child: Text(_isDownloading ? 'Downloading...' : 'Start Download'),
                    ),
                  ],
                ),
              ),
            ),
          ],
        ),
      ),
    );
  }

  @override
  void dispose() {
    _circularController.dispose();
    _linearController.dispose();
    super.dispose();
  }
}
```

```dart
          value: _customProgress,
          backgroundColor: Colors.grey[300],
          valueColor: AlwaysStoppedAnimation<Color>(Colors.blue),
        ),
      ],
    );
  }
```

## Advanced Progress Patterns

### Complex Progress Implementation

```dart
class AdvancedProgressExample extends StatefulWidget {
  @override
  _AdvancedProgressExampleState createState() => _AdvancedProgressExampleState();
}

class _AdvancedProgressExampleState extends State<AdvancedProgressExample>
    with TickerProviderStateMixin {
  late AnimationController _controller;
  late Animation<double> _animation;
  
  List<UploadTask> _uploadTasks = [];
  int _completedTasks = 0;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(
      duration: Duration(seconds: 2),
      vsync: this,
    );
    _animation = CurvedAnimation(parent: _controller, curve: Curves.easeInOut);
  }

  void _startMultipleUploads() {
    setState(() {
      _uploadTasks = [
        UploadTask('Document.pdf', 0.0),
        UploadTask('Image.jpg', 0.0),
        UploadTask('Video.mp4', 0.0),
        UploadTask('Archive.zip', 0.0),
      ];
      _completedTasks = 0;
    });

    for (int i = 0; i < _uploadTasks.length; i++) {
      _simulateUpload(i);
    }
  }

  void _simulateUpload(int index) {
    Timer.periodic(Duration(milliseconds: 50), (timer) {
      setState(() {
        _uploadTasks[index].progress += 0.02;
        if (_uploadTasks[index].progress >= 1.0) {
          _uploadTasks[index].progress = 1.0;
          _uploadTasks[index].isCompleted = true;
          _completedTasks++;
          timer.cancel();
        }
      });
    });
  }

  @override
  Widget build(BuildContext context) {
    double overallProgress = _uploadTasks.isEmpty 
        ? 0.0 
        : _uploadTasks.map((task) => task.progress).reduce((a, b) => a + b) / _uploadTasks.length;

    return Scaffold(
      appBar: AppBar(
        title: Text('Advanced Progress Patterns'),
        backgroundColor: Colors.deepOrange,
      ),
      body: Padding(
        padding: EdgeInsets.all(16.0),
        child: Column(
          children: [
            // Overall Progress
            Card(
              child: Padding(
                padding: EdgeInsets.all(20),
                child: Column(
                  crossAxisAlignment: CrossAxisAlignment.start,
                  children: [
                    Text(
                      'Multiple File Upload Progress',
                      style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
                    ),
                    SizedBox(height: 20),
                    Row(
                      mainAxisAlignment: MainAxisAlignment.spaceBetween,
                      children: [
                        Text('Overall Progress'),
                        Text('$_completedTasks/${_uploadTasks.length} completed'),
                      ],
                    ),
                    SizedBox(height: 10),
                    LinearProgressIndicator(
                      value: overallProgress,
                      backgroundColor: Colors.grey[300],
                      valueColor: AlwaysStoppedAnimation<Color>(Colors.deepOrange),
                      minHeight: 8,
                    ),
                    SizedBox(height: 10),
                    Text('${(overallProgress * 100).toInt()}% complete'),
                  ],
                ),
              ),
            ),
            
            SizedBox(height: 20),
            
            // Individual Task Progress
            Expanded(
              child: ListView.builder(
                itemCount: _uploadTasks.length,
                itemBuilder: (context, index) {
                  final task = _uploadTasks[index];
                  return Card(
                    margin: EdgeInsets.only(bottom: 10),
                    child: Padding(
                      padding: EdgeInsets.all(16),
                      child: Column(
                        crossAxisAlignment: CrossAxisAlignment.start,
                        children: [
                          Row(
                            children: [
                              Icon(
                                task.isCompleted ? Icons.check_circle : Icons.file_present,
                                color: task.isCompleted ? Colors.green : Colors.grey,
                              ),
                              SizedBox(width: 10),
                              Expanded(
                                child: Text(
                                  task.fileName,
                                  style: TextStyle(fontWeight: FontWeight.bold),
                                ),
                              ),
                              Text(
                                task.isCompleted 
                                    ? 'Complete' 
                                    : '${(task.progress * 100).toInt()}%',
                                style: TextStyle(
                                  color: task.isCompleted ? Colors.green : Colors.grey[600],
                                ),
                              ),
                            ],
                          ),
                          SizedBox(height: 10),
                          LinearProgressIndicator(
                            value: task.progress,
                            backgroundColor: Colors.grey[300],
                            valueColor: AlwaysStoppedAnimation<Color>(
                              task.isCompleted ? Colors.green : Colors.blue,
                            ),
                            minHeight: 6,
                          ),
                        ],
                      ),
                    ),
                  );
                },
              ),
            ),
            
            SizedBox(height: 20),
            
            ElevatedButton(
              onPressed: _uploadTasks.isEmpty || _completedTasks == _uploadTasks.length
                  ? _startMultipleUploads
                  : null,
              child: Text(
                _uploadTasks.isEmpty 
                    ? 'Start Upload' 
                    : _completedTasks == _uploadTasks.length
                        ? 'Upload Again'
                        : 'Uploading...',
              ),
              style: ElevatedButton.styleFrom(
                backgroundColor: Colors.deepOrange,
                minimumSize: Size(double.infinity, 48),
              ),
            ),
          ],
        ),
      ),
    );
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }
}

class UploadTask {
  final String fileName;
  double progress;
  bool isCompleted;

  UploadTask(this.fileName, this.progress, {this.isCompleted = false});
}
```

## Tips and Best Practices

### 1. **Choose the Right Type**
- Use **CircularProgressIndicator** for unknown duration tasks
- Use **LinearProgressIndicator** for known progress with percentages
- Consider custom indicators for specific use cases

### 2. **Performance Optimization**
- Use `const` constructors when possible
- Avoid unnecessary setState calls
- Use AnimationController for smooth animations
- Dispose controllers properly to prevent memory leaks

### 3. **User Experience**
- Always provide feedback about what's happening
- Show percentage for determinate progress
- Use appropriate colors (green for success, red for errors)
- Implement proper loading states

### 4. **Accessibility**
- Provide semantic labels for screen readers
- Announce progress updates
- Ensure sufficient color contrast
- Consider users with motion sensitivity

### 5. **Visual Design**
- Match your app's theme and color scheme
- Use consistent sizing across the app
- Consider dark/light theme support
- Implement smooth transitions
