# Text in Flutter Material UI

## What is Text Widget?

The **Text** widget is one of the most fundamental and frequently used widgets in Flutter. It displays a string of text with optional styling. The Text widget is highly customizable and supports various formatting options including fonts, colors, alignment, decoration, and responsive behavior. It's the primary way to display textual content in Flutter applications.

## Key Features

- **Rich Text Support**: Supports various text styles and formatting
- **Material Typography**: Integrates with Material Design typography themes
- **Responsive**: Automatically handles text scaling and overflow
- **Internationalization**: Supports multiple languages and text directions
- **Accessibility**: Built-in accessibility features for screen readers
- **Performance**: Optimized rendering for smooth scrolling and animations

class StrokeTextPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final textPainter = TextPainter(
      text: TextSpan(
        text: 'OUTLINED TEXT',
        style: TextStyle(
          fontSize: 28,
          fontWeight: FontWeight.bold,
          foreground: Paint()
            ..style = PaintingStyle.stroke
            ..strokeWidth = 2
            ..color = Colors.blue,
        ),
      ),
      textDirection: TextDirection.ltr,
    );
    
    textPainter.layout();
    
    final offset = Offset(
      (size.width - textPainter.width) / 2,
      (size.height - textPainter.height) / 2,
    );
    
    textPainter.paint(canvas, offset);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

## Tips and Best Practices

1. **Performance Optimization**:
   - Use `const` constructors when text style doesn't change
   - Avoid creating new TextStyle objects unnecessarily
   - Use theme typography for consistency

2. **Accessibility**:
   - Ensure sufficient color contrast
   - Support text scaling for users with visual impairments
   - Use semantic labels for screen readers

3. **Internationalization**:
   - Support right-to-left languages
   - Use proper locale settings
   - Handle text direction correctly

4. **Typography Hierarchy**:
   - Use Material Design typography scales
   - Maintain consistent font weights and sizes
   - Create clear visual hierarchy

5. **Responsive Design**:
   - Adapt text sizes for different screen sizes
   - Handle text overflow gracefully
   - Consider reading distance and device type
