# Advanced Features in Flutter

Advanced features in Flutter are like **special superpowers** for your app - they make your app feel professional, smooth, and delightful to use.

## What are Advanced Features? (Simple Explanation)

Think of your Flutter app like a **movie**:

```
🎬 BASIC APP (Like a Simple Video)
┌─────────────────────────────────────────────┐
│  📱 Screen 1 → 📱 Screen 2 → 📱 Screen 3    │
│  Just switching between pages               │
│  No smooth transitions                     │
│  No special effects                        │
└─────────────────────────────────────────────┘

🎭 ADVANCED APP (Like a Hollywood Movie)
┌─────────────────────────────────────────────┐
│  📱 Screen slides in smoothly ✨            │
│  📱 Icons bounce and spin 🎪               │
│  📱 Custom transitions between pages 🌟     │
│  📱 Interactive animations 🎨              │
└─────────────────────────────────────────────┘
```

**Advanced Features Include:**
- **Advanced Navigation**: Complex routing, nested navigation, custom transitions
- **Animations**: Making things move, bounce, fade, and transform smoothly
- **Custom Painting**: Drawing custom graphics and shapes

## 1. Advanced Navigation

Advanced navigation is like having a **smart GPS system** for your app that can handle complex routes and transitions.

### Basic Navigation vs Advanced Navigation

```dart
// 🚗 BASIC NAVIGATION: Simple point-to-point
Navigator.push(context, MaterialPageRoute(builder: (context) => NextPage()));

// 🚁 ADVANCED NAVIGATION: Complex routing with names, parameters, guards
Navigator.pushNamed(context, '/profile/123', arguments: {'userId': 123});
```

```dart
// 📋 STEP 1: Define route names
class AppRoutes {
  static const String home = '/';
  static const String login = '/login';
  static const String profile = '/profile';
  static const String userDetail = '/user/:userId';
  static const String settings = '/settings';
  static const String notification = '/notification';
}

// 📋 STEP 2: Create route generator
class RouteGenerator {
  static Route<dynamic> generateRoute(RouteSettings settings) {
    final args = settings.arguments;
    
    switch (settings.name) {
      case AppRoutes.home:
        return MaterialPageRoute(builder: (_) => HomeScreen());
        
      case AppRoutes.login:
        return MaterialPageRoute(builder: (_) => LoginScreen());
        
      case AppRoutes.profile:
        return MaterialPageRoute(builder: (_) => ProfileScreen());
        
      case AppRoutes.settings:
        return MaterialPageRoute(builder: (_) => SettingsScreen());
        
      default:
        // Handle dynamic routes like /user/123
        if (settings.name!.startsWith('/user/')) {
          final userId = settings.name!.split('/')[2];
          return MaterialPageRoute(
            builder: (_) => UserDetailScreen(userId: userId),
          );
        }
        
        // Unknown route
        return MaterialPageRoute(
          builder: (_) => Scaffold(
            appBar: AppBar(title: Text('Page Not Found')),
            body: Center(
              child: Column(
                mainAxisAlignment: MainAxisAlignment.center,
                children: [
                  Icon(Icons.error, size: 64, color: Colors.red),
                  SizedBox(height: 16),
                  Text('404 - Page Not Found'),
                  SizedBox(height: 16),
                  ElevatedButton(
                    onPressed: () => Navigator.pushNamedAndRemoveUntil(
                      _, AppRoutes.home, (route) => false,
                    ),
                    child: Text('Go Home'),
                  ),
                ],
              ),
            ),
          ),
        );
    }
  }
}
```

### 1.2 Custom Page Transitions

```dart
// 🎭 CUSTOM TRANSITIONS: Make navigation feel special

// Slide transition
Route _createSlideRoute(Widget page) {
  return PageRouteBuilder(
    pageBuilder: (context, animation, secondaryAnimation) => page,
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      const begin = Offset(1.0, 0.0); // Slide from right
      const end = Offset.zero;
      const curve = Curves.easeInOutCubic;
      
      var tween = Tween(begin: begin, end: end).chain(
        CurveTween(curve: curve),
      );
      
      return SlideTransition(
        position: animation.drive(tween),
        child: child,
      );
    },
    transitionDuration: Duration(milliseconds: 600),
  );
}

// Fade transition
Route _createFadeRoute(Widget page) {
  return PageRouteBuilder(
    pageBuilder: (context, animation, secondaryAnimation) => page,
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      return FadeTransition(opacity: animation, child: child);
    },
    transitionDuration: Duration(milliseconds: 400),
  );
}

// Scale transition
Route _createScaleRoute(Widget page) {
  return PageRouteBuilder(
    pageBuilder: (context, animation, secondaryAnimation) => page,
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      return ScaleTransition(
        scale: Tween<double>(begin: 0.0, end: 1.0).animate(
          CurvedAnimation(parent: animation, curve: Curves.elasticOut),
        ),
        child: child,
      );
    },
    transitionDuration: Duration(milliseconds: 800),
  );
}

// Rotation transition
Route _createRotationRoute(Widget page) {
  return PageRouteBuilder(
    pageBuilder: (context, animation, secondaryAnimation) => page,
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      return RotationTransition(
        turns: Tween<double>(begin: 0.0, end: 1.0).animate(animation),
        child: FadeTransition(opacity: animation, child: child),
      );
    },
    transitionDuration: Duration(milliseconds: 600),
  );
}
```

### 2.2 Explicit Animations (More Control)

```dart
// 🟡 EXPLICIT ANIMATIONS: You control every detail!

class ExplicitAnimationsDemo extends StatefulWidget {
  @override
  _ExplicitAnimationsDemoState createState() => _ExplicitAnimationsDemoState();
}

class _ExplicitAnimationsDemoState extends State<ExplicitAnimationsDemo>
    with TickerProviderStateMixin {
  
  late AnimationController _rotationController;
  late AnimationController _scaleController;
  late AnimationController _slideController;
  late AnimationController _colorController;
  
  late Animation<double> _rotationAnimation;
  late Animation<double> _scaleAnimation;
  late Animation<Offset> _slideAnimation;
  late Animation<Color?> _colorAnimation;
  
  @override
  void initState() {
    super.initState();
    
    // 🎛️ CREATE ANIMATION CONTROLLERS
    _rotationController = AnimationController(
      duration: Duration(seconds: 2),
      vsync: this,
    );
    
    _scaleController = AnimationController(
      duration: Duration(milliseconds: 800),
      vsync: this,
    );
    
    _slideController = AnimationController(
      duration: Duration(milliseconds: 600),
      vsync: this,
    );
    
    _colorController = AnimationController(
      duration: Duration(seconds: 3),
      vsync: this,
    );
    
    // 🎨 CREATE ANIMATIONS WITH TWEENS
    _rotationAnimation = Tween<double>(
      begin: 0.0,
      end: 2.0, // 2 full rotations
    ).animate(CurvedAnimation(
      parent: _rotationController,
      curve: Curves.easeInOut,
    ));
    
    _scaleAnimation = Tween<double>(
      begin: 0.5,
      end: 1.5,
    ).animate(CurvedAnimation(
      parent: _scaleController,
      curve: Curves.elasticOut,
    ));
    
    _slideAnimation = Tween<Offset>(
      begin: Offset(-1.0, 0.0),
      end: Offset(1.0, 0.0),
    ).animate(CurvedAnimation(
      parent: _slideController,
      curve: Curves.easeInOutCubic,
    ));
    
    _colorAnimation = ColorTween(
      begin: Colors.blue,
      end: Colors.red,
    ).animate(_colorController);
    
    // 🔄 REPEAT SOME ANIMATIONS
    _rotationController.repeat();
    _colorController.repeat(reverse: true);
  }
  
  @override
  void dispose() {
    _rotationController.dispose();
    _scaleController.dispose();
```

class WaveCirclePainter extends CustomPainter {
  final double animationValue;
  
  WaveCirclePainter(this.animationValue);
  
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()
      ..color = Colors.blue.withOpacity(0.3)
      ..style = PaintingStyle.stroke
      ..strokeWidth = 2;
    
    final center = Offset(size.width / 2, size.height / 2);
    
    // Draw multiple expanding circles
    for (int i = 0; i < 5; i++) {
      final radius = (animationValue + i * 0.2) * size.width / 3;
      final opacity = 1.0 - ((animationValue + i * 0.2) % 1.0);
      
      paint.color = Colors.blue.withOpacity(opacity * 0.5);
      
      if (radius <= size.width / 2) {
        canvas.drawCircle(center, radius, paint);
      }
    }
  }
  
  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => true;
}

// 📊 BAR CHART PAINTER
class BarChartPainter extends CustomPainter {
  final double animationValue;
  
  BarChartPainter(this.animationValue);
  
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()..style = PaintingStyle.fill;
    
    final data = [0.8, 0.6, 0.9, 0.4, 0.7, 0.5]; // Bar heights (0-1)
    final colors = [
      Colors.red,
      Colors.blue,
      Colors.green,
      Colors.orange,
      Colors.purple,
      Colors.cyan,
    ];
    
    final barWidth = size.width / (data.length * 2);
    final maxHeight = size.height - 40;
    
    for (int i = 0; i < data.length; i++) {
      final barHeight = data[i] * maxHeight * animationValue;
      final x = (i * 2 + 1) * barWidth;
      final y = size.height - 20 - barHeight;
      
      paint.color = colors[i];
      
      // Draw bar
      canvas.drawRect(
        Rect.fromLTWH(x, y, barWidth, barHeight),
        paint,
      );
      
      // Draw value text
      final textPainter = TextPainter(
        text: TextSpan(
          text: '${(data[i] * 100).round()}%',
          style: TextStyle(color: Colors.black, fontSize: 12),
        ),
        textDirection: TextDirection.ltr,
      );
      textPainter.layout();
      textPainter.paint(
        canvas,
        Offset(x + barWidth / 2 - textPainter.width / 2, y - 20),
      );
    }
  }
  
  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => true;
}

// ⭐ STARRY NIGHT PAINTER
class StarryNightPainter extends CustomPainter {
  final double animationValue;
  
  StarryNightPainter(this.animationValue);
  
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()..style = PaintingStyle.fill;
    
    // Generate "random" stars (using deterministic positions)
    for (int i = 0; i < 50; i++) {
      final x = (i * 37) % size.width.toInt().clamp(0, size.width.toInt());
      final y = (i * 23) % size.height.toInt().clamp(0, size.height.toInt());
      
      // Animate opacity for twinkling effect
      final twinkle = (animationValue + i * 0.1) % 1.0;
      final opacity = 0.3 + 0.7 * (0.5 + 0.5 * math.sin(twinkle * 2 * math.pi));
      
      paint.color = Colors.white.withOpacity(opacity);
      
      // Draw star shape
      _drawStar(canvas, Offset(x.toDouble(), y.toDouble()), 3, paint);
    }
  }
  
  void _drawStar(Canvas canvas, Offset center, double radius, Paint paint) {
    final path = Path();
    final spikes = 5;
    final outerRadius = radius;
    final innerRadius = radius * 0.4;
    
    for (int i = 0; i < spikes * 2; i++) {
      final angle = (i * math.pi) / spikes;
      final currentRadius = i.isEven ? outerRadius : innerRadius;
      final x = center.dx + currentRadius * math.cos(angle);
      final y = center.dy + currentRadius * math.sin(angle);
      
      if (i == 0) {
        path.moveTo(x, y);
      } else {
        path.lineTo(x, y);
      }
    }
    path.close();
    canvas.drawPath(path, paint);
  }
  
  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => true;
}

// 🌸 FLOWER PAINTER
class FlowerPainter extends CustomPainter {
  final double animationValue;
  
  FlowerPainter(this.animationValue);
  
  @override
  void paint(Canvas canvas, Size size) {
    final center = Offset(size.width / 2, size.height / 2);
    
    // Draw stem
    final stemPaint = Paint()
      ..color = Colors.green
      ..strokeWidth = 4
      ..style = PaintingStyle.stroke;
    
    final stemHeight = size.height * 0.6 * animationValue;
    canvas.drawLine(
      Offset(center.dx, size.height),
      Offset(center.dx, size.height - stemHeight),
      stemPaint,
    );
    
    // Draw petals (only if stem is grown)
    if (animationValue > 0.5) {
      final petalProgress = (animationValue - 0.5) * 2;
      final petalPaint = Paint()
        ..color = Colors.pink
        ..style = PaintingStyle.fill;
      
      // Draw 8 petals around center
      for (int i = 0; i < 8; i++) {
        final angle = (i * 2 * math.pi) / 8;
        final petalLength = 30 * petalProgress;
        final petalWidth = 15 * petalProgress;
        
        canvas.save();
        canvas.translate(center.dx, center.dy - stemHeight + 20);
        canvas.rotate(angle);
        
        // Draw petal as ellipse
        canvas.drawOval(
          Rect.fromCenter(
            center: Offset(petalLength / 2, 0),
            width: petalLength,
            height: petalWidth,
          ),
          petalPaint,
        );
        
        canvas.restore();
      }
      
      // Draw center
      final centerPaint = Paint()
        ..color = Colors.yellow
        ..style = PaintingStyle.fill;
      
      canvas.drawCircle(
        Offset(center.dx, center.dy - stemHeight + 20),
        10 * petalProgress,
        centerPaint,
      );
    }
  }
  
  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => true;
}

## Summary

### Advanced Navigation Key Points:
1. **Named Routes**: Organize navigation with string routes
2. **Route Generation**: Handle dynamic routes and parameters
3. **Custom Transitions**: Create unique page transitions
4. **Nested Navigation**: TabBar with individual navigation stacks
5. **Modal Navigation**: Dialogs, bottom sheets, full-screen modals

### Animation Key Points:
1. **Implicit Animations**: Easy to use, automatic transitions
   - `AnimatedContainer`, `AnimatedOpacity`, `AnimatedPositioned`
2. **Explicit Animations**: Full control with `AnimationController`
   - Custom timing, curves, and complex sequences
3. **CustomPainter**: Draw custom graphics and shapes
   - Unlimited creative possibilities

### When to Use Each:

**Implicit Animations:**
- Simple property changes
- Quick prototyping
- Basic UI feedback

**Explicit Animations:**
- Complex sequences
- Precise timing control
- Multiple coordinated animations

**CustomPainter:**
- Custom graphics
- Data visualizations
- Unique visual elements

### Best Practices:
1. **Performance**: Dispose controllers properly
2. **User Experience**: Don't overuse animations
3. **Accessibility**: Consider users with motion sensitivity
4. **Consistency**: Use similar animations throughout your app

Advanced features make your Flutter app feel **professional and delightful** to use!
